# OctoAcme Project Management Docs

This repository documents the project management processes used at OctoAcme to enable consistent, effective delivery and knowledge sharing across teams.

## Summary of Project Management Processes

- **Customer-first, iterative delivery:** Projects are driven by customer value and delivered in small, testable increments.
- **Clear ownership:** Each project is led by a dedicated Project Manager (PM) and Product Lead, with distinct responsibilities.
- **Data-informed planning and retrospective cycles:** Metrics, planning, and structured feedback drive continuous improvement and smart prioritization.
- **Standardized artifacts:** Core deliverables include a one-pager, sprints, Definition of Done, risk register, and release plan.
- **Transparency and risk management:** Work, risks, status, and communications are proactively managed and visible.
- **Continuous improvement:** After each milestone or incident, retrospectives are conducted and learnings are adopted.
